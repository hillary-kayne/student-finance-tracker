Currency Rates

Base currency is USD. If you spend in other currencies like RWF (Rwandan Franc) or EUR (Euro), set the exchange rate below so the app can convert amounts for you. For example, if 1 USD = 1,300 RWF, type 1300 in the RWF field. You update these manually — no internet needed.

SaveStudent is built for students across Africa and the world — whether you spend in RWF, KES, GHS, NGN, or USD, you belong here.
